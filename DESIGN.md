# DESIGN.md

# ForgeMind Design System

## Emotional Feel

ForgeMind should feel:
- calm
- grounded
- private
- emotionally safe
- modern
- premium

Avoid:
- clinical hospital feel
- childish wellness branding
- overstimulating interfaces

---

# Theme

Dark mode first.

Palette:
- charcoal
- black
- muted gray
- subtle blue accents

Avoid:
- neon colors
- excessive gradients
- flashy animation

---

# Navigation

Bottom tabs:

1. Home
2. Talk
3. Reset
4. Progress
5. Profile

---

# Chat UX

Primary product experience.

Requirements:
- minimal UI
- large microphone button
- low distraction
- generous spacing
- smooth message transitions

---

# Voice UX

MVP:
- Tap-to-Talk
- AAC .m4a uploads
- POST API

Future:
- realtime conversation
- websocket streaming
- turn detection

Voice interactions should feel:
- calm
- intentional
- reflective

---

# Typography

Requirements:
- readable
- spacious
- low cognitive load

Avoid:
- clutter
- giant walls of text
- dense layouts

---

# Home Screen

Contains:
- greeting
- quick emotional actions
- daily insights
- memory-based reflections

Quick actions:
- Talk it out
- Calm down
- Angry
- Overwhelmed
- Can't sleep

---

# Reset Tools

Reset flows should:
- be short
- practical
- emotionally grounding
- calming

Avoid:
- long meditation experiences
- overwhelming advice

---

# Emotional UX Principles

The user should feel:
- understood
- calmer
- less alone
- more clear-headed
- emotionally safer
