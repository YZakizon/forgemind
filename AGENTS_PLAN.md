# ForgeMind Product Plan

## Vision

Build a calm, private AI emotional support platform for men.

ForgeMind should feel like:
- a grounded emotional support system
- a calm mental clarity tool
- a reflective coach

NOT:
- a therapy office
- a social media app
- a generic chatbot

---

# Product Goals

Primary goals:
- emotional clarity
- stress reduction
- loneliness support
- healthier reflection
- safer emotional processing

Secondary goals:
- habit consistency
- emotional awareness
- sleep improvement
- communication improvement

---

# MVP Scope

## Mobile Screens

1. Onboarding
2. Home
3. AI Chat
4. Reset Tools
5. Progress
6. Settings

# Onboarding
